# 历史见证者

## 南京大屠杀见证者

### 拉贝
- **全名**：约翰·拉贝
- **生卒**：1882年11月23日-1950年1月5日
- **国籍**：德国
- **身份**：西门子公司驻南京代表
- **主要贡献**：
  - 建立南京安全区
  - 保护25万中国难民
  - 撰写《拉贝日记》
- **历史评价**：中国的辛德勒
- **名言**："我不相信这是上帝的旨意"
