# 盟国将领

支援中国抗战的盟军军事人员。

## 美国

### 陈纳德
- **全名**：克莱尔·李·陈纳德
- **生卒**：1893年9月6日-1958年7月27日
- **国籍**：美国
- **来华时间**：1937年
- **职务**：中国空军美国志愿援华航空队（飞虎队）队长
- **主要贡献**：
  - 组建飞虎队
  - 协助中国空军作战
  - 打击日军空中力量
- **历史评价**：中国空军的救星

### 史迪威
- **全名**：约瑟夫·沃伦·史迪威
- **生卒**：1883年3月19日-1946年10月12日
- **国籍**：美国
- **来华时间**：1942年
- **职务**：中国战区参谋长
- **主要贡献**：
  - 指挥中国远征军作战
  - 协调中美军事合作
- **历史评价**：中国人民的朋友

### 魏德迈
- **全名**：阿尔伯特·科蒂·魏德迈
- **生卒**：1897年7月31日-1989年12月12日
- **国籍**：美国
- **来华时间**：1944年
- **职务**：中国战区美军司令
- **主要贡献**：
  - 协助中国军队作战
  - 提供军事援助
- **历史评价**：中美军事合作的重要人物
