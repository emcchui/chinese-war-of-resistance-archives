# 史迪威

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 约瑟夫·沃伦·史迪威 |
| 生卒 | 1883年3月19日-1946年10月12日 |
| 国籍 | 美国 |
| 来华时间 | 1942年 |
| 职务 | 中国战区参谋长 |

## 生平简介

史迪威是美国陆军将领，中国战区美军司令。

## 主要贡献

### 指挥作战
- 指挥中国远征军作战
- 协调中美军事合作
- 参与制定作战计划

### 军事援助
- 协助中国军队训练
- 提供美式装备
- 改善后勤保障

## 主要战役

- 中国远征军入缅作战
- 滇西反攻

## 历史评价

- 中国人民的朋友
- 中美军事合作的重要人物
