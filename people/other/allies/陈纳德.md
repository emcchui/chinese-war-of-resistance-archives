# 陈纳德

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 克莱尔·李·陈纳德 |
| 生卒 | 1893年9月6日-1958年7月27日 |
| 国籍 | 美国 |
| 来华时间 | 1937年 |
| 职务 | 飞虎队队长 |

## 生平简介

陈纳德是美国空军将领，飞虎队的创建者。

## 主要贡献

### 组建飞虎队
- 组建中国空军美国志愿援华航空队（飞虎队）
- 协助中国空军作战
- 打击日军空中力量

### 空中作战
- 指挥飞虎队作战
- 击落大量日军飞机
- 保护中国后方

## 历史评价

- 中国空军的救星
- 中国人民的朋友

## 大事年表

| 时间 | 事件 |
|------|------|
| 1893年9月6日 | 出生于美国得克萨斯州 |
| 1917年 | 入美国陆军航空队 |
| 1920年 | 成为战斗机飞行员 |
| 1923年 | 任飞行教官 |
| 1930年 | 入美国陆军航空队学校 |
| 1935年 | 出版《防御性追击的作用》 |
| 1937年 | 来到中国 |
| 1937年 | 任中国空军顾问 |
| 1941年 | 组建飞虎队 |
| 1941年 | 飞虎队成立 |
| 1942年 | 飞虎队改编为美国陆军航空队 |
| 1943年 | 任美国陆军航空队中国特遣队司令 |
| 1944年 | 任美国陆军航空队驻华司令 |
| 1945年 | 日本投降 |
| 1946年 | 回到美国 |
| 1958年7月27日 | 在美国逝世 |
