# 杨靖宇

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 杨靖宇 |
| 原名 | 马尚德 |
| 生卒 | 1905年2月13日-1940年2月23日 |
| 籍贯 | 河南确山 |
| 职位 | 东北抗日联军第一路军总司令 |
| 殉国地点 | 吉林濛江 |

## 生平简介

杨靖宇是东北抗日联军的主要领导人，著名的抗日英雄。

## 主要贡献

### 东北抗战
- 领导东北抗日联军
- 开展游击战争
- 坚持东北抗战

### 抗联建设
- 创建抗联队伍
- 发展抗日武装
- 建立根据地

## 殉国经过

### 最后的战斗
- 1940年2月，杨靖宇被日军包围
- 在濛江县壮烈殉国
- 日军解剖其遗体，发现胃中只有草根和棉絮

## 历史评价

- 东北抗日英雄
- 宁死不屈的民族气节
