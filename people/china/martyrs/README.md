# 殉国英烈

为抗击日本侵略而英勇牺牲的中国英雄。

## 殉国将领

### 佟麟阁
- **职位**：29军副军长
- **生卒**：1892年-1937年7月28日
- **籍贯**：河北高阳
- **殉国地点**：北平南苑
- **事迹**：
  - 全面抗战爆发后牺牲的第一位高级将领
  - 在南苑保卫战中壮烈殉国
- **名言**："国家多难，军人应当马革裹尸，以死报国"

### 赵登禹
- **职位**：29军132师师长
- **生卒**：1898年-1937年7月28日
- **籍贯**：山东菏泽
- **殉国地点**：北平南苑
- **事迹**：
  - 与佟麟阁同时殉国
  - 在南苑保卫战中英勇作战
- **名言**："军人抗战有死无生，卢沟桥就是我们的坟墓"

### 王铭章
- **职位**：第122师师长
- **生卒**：1893年-1938年3月17日
- **籍贯**：四川新都
- **殉国地点**：山东滕县
- **事迹**：
  - 台儿庄战役前保卫滕县
  - 与城共存亡，壮烈殉国
- **名言**："现在正是军人报国的时候"

### 张自忠
- **职位**：第33集团军总司令
- **生卒**：1891年8月11日-1940年5月16日
- **籍贯**：山东临清
- **殉国地点**：湖北南瓜店
- **事迹**：
  - 抗战中牺牲的最高将领
  - 在枣宜会战中壮烈殉国
- **名言**："为国家民族死之决心，海不清，石不烂，决不半点改变"

### 戴安澜
- **职位**：中国远征军第200师师长
- **生卒**：1904年11月25日-1942年5月26日
- **籍贯**：安徽无为
- **殉国地点**：缅甸北部
- **事迹**：
  - 中国远征军著名将领
  - 同古保卫战英雄
- **名言**："为国战死，事极光荣"

### 左权
- **职位**：八路军副参谋长
- **生卒**：1905年3月15日-1942年5月25日
- **籍贯**：湖南醴陵
- **殉国地点**：山西辽县
- **事迹**：
  - 八路军在抗战中牺牲的最高将领
  - 在反扫荡作战中壮烈殉国
- **名言**："我牺牲了我的一切幸福，为我的事业奋斗"

## 英雄群体

### 八女投江
- **时间**：1938年10月
- **地点**：黑龙江乌斯浑河
- **人物**：冷云、杨贵珍、郭桂琴等八名女战士
- **事迹**：
  - 抗联八名女战士为掩护部队突围
  - 将敌人引向自己，投江殉国
- **精神**：宁死不屈的英雄气概

### 狼牙山五壮士
- **时间**：1941年8月
- **地点**：河北易县狼牙山
- **人物**：马宝玉、葛振林、宋学义、胡德林、胡福才
- **事迹**：
  - 为掩护群众和主力转移
  - 将敌人引上绝路，跳崖殉国
- **精神**：宁死不屈的民族气节

### 雁翎队
- **时间**：1939年-1945年
- **地点**：河北白洋淀
- **主要贡献**：
  - 在水上与日军作战
  - 保护群众，打击敌人
- **精神**：灵活机动的游击战术

## 精神传承

这些英烈的抗战精神值得我们永远铭记：
1. 天下兴亡、匹夫有责的爱国情怀
2. 视死如归、宁死不屈的民族气节
3. 不畏强暴、血战到底的英雄气概
4. 百折不挠、坚忍不拔的必胜信念
