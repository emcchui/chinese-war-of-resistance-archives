# 赵尚志

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 赵尚志 |
| 生卒 | 1908年10月26日-1942年2月12日 |
| 籍贯 | 辽宁朝阳 |
| 职位 | 东北抗日联军第三军军长 |
| 殉国地点 | 黑龙江珠河 |

## 生平简介

赵尚志是东北抗日联军的主要领导人之一。

## 主要贡献

### 东北抗战
- 领导东北抗日联军
- 开展游击战争
- 坚持东北抗战

### 抗联建设
- 创建抗联队伍
- 发展抗日武装

## 殉国经过

### 最后的战斗
- 1942年2月，在战斗中壮烈牺牲

## 历史评价

- 东北抗日英雄
- 宁死不屈的民族气节
