# 狼牙山五壮士

## 基本信息

| 项目 | 内容 |
|------|------|
| 事件时间 | 1941年8月 |
| 地点 | 河北易县狼牙山 |
| 人物 | 马宝玉、葛振林、宋学义、胡德林、胡福才 |
| 部队 | 八路军晋察冀军区一分区一团七连 |

## 事件经过

### 掩护群众
- 1941年8月，日军对晋察冀根据地进行大扫荡
- 七连六班负责掩护群众和主力转移

### 引敌上山
- 五名战士将敌人引上狼牙山
- 与敌人激战

### 跳崖殉国
- 在弹尽粮绝的情况下
- 五名战士宁死不屈，跳崖殉国
- 马宝玉、胡德林、胡福才壮烈牺牲
- 葛振林、宋学义被山腰树枝挂住，幸免于难
