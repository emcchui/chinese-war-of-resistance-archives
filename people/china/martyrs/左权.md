# 左权

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 左权 |
| 原名 | 左纪权 |
| 生卒 | 1905年3月15日-1942年5月25日 |
| 籍贯 | 湖南醴陵 |
| 职位 | 八路军副参谋长 |
| 殉国地点 | 山西辽县 |

## 生平简介

左权是八路军在抗战中牺牲的最高将领。

## 主要贡献

### 军事指挥
- 协助指挥百团大战
- 参与制定作战计划
- 领导根据地建设

### 军事理论
- 撰写军事著作
- 总结游击战术

## 殉国经过

### 反扫荡作战
- 1942年5月，日军对太行根据地进行大扫荡
- 5月25日，在指挥突围时壮烈殉国

## 历史评价

- 八路军在抗战中牺牲的最高将领
- 杰出的军事家

## 名言

> "我牺牲了我的一切幸福，为我的事业奋斗"
