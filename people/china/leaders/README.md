# 中国领导人物

抗日战争时期的中国领导层。

## 国民政府领导

### 蒋介石
- **职位**：国民政府军事委员会委员长
- **生卒**：1887年10月31日-1975年4月5日
- **籍贯**：浙江奉化
- **主要作用**：
  - 领导国民政府进行抗战
  - 指挥正面战场作战
  - 迁都重庆，坚持长期抗战
- **主要决策**：
  - 淞沪会战
  - 武汉会战
  - 豫湘桂战役
- **历史评价**：坚持抗战到底，但战略战术有失误

### 林森
- **职位**：国民政府主席
- **生卒**：1868年2月11日-1943年8月1日
- **籍贯**：福建闽侯
- **主要作用**：
  - 国家元首，主持国政
  - 维护抗战团结
- **历史评价**：坚持抗战立场

### 宋子文
- **职位**：行政院院长
- **生卒**：1894年12月4日-1971年4月26日
- **籍贯**：广东文昌
- **主要作用**：
  - 负责外交事务
  - 争取国际援助
- **历史评价**：为抗战争取国际支持

### 孔祥熙
- **职位**：行政院副院长
- **生卒**：1880年9月11日-1967年8月16日
- **籍贯**：山西太谷
- **主要作用**：
  - 负责财政事务
  - 筹措抗战经费
- **历史评价**：为抗战提供财政支持

## 中共领导

### 毛泽东
- **职位**：中共中央主席
- **生卒**：1893年12月26日-1976年9月9日
- **籍贯**：湖南湘潭
- **主要作用**：
  - 提出持久战理论
  - 领导敌后抗战
  - 建立抗日民族统一战线
- **主要著作**：
  - 《论持久战》
  - 《论新阶段》
  - 《抗日游击战争的战略问题》
- **历史评价**：正确领导敌后战场，为抗战胜利奠定基础

### 周恩来
- **职位**：中共中央军委副主席
- **生卒**：1898年3月5日-1976年1月8日
- **籍贯**：浙江绍兴
- **主要作用**：
  - 负责统战工作
  - 推动国共合作
  - 协调各方关系
- **历史评价**：为团结抗战作出重要贡献

### 朱德
- **职位**：八路军总司令
- **生卒**：1886年12月1日-1976年7月6日
- **籍贯**：四川仪陇
- **主要作用**：
  - 指挥八路军敌后抗战
  - 参与百团大战指挥
- **历史评价**：敌后战场主要指挥者

### 彭德怀
- **职位**：八路军副总司令
- **生卒**：1898年10月24日-1974年11月29日
- **籍贯**：湖南湘潭
- **主要作用**：
  - 指挥百团大战
  - 领导华北敌后抗战
- **历史评价**：英勇善战，指挥果断

## 其他重要人物

### 张学良
- **职位**：原东北边防军司令长官
- **生卒**：1901年6月3日-2001年10月14日
- **籍贯**：辽宁台安
- **主要事件**：
  - 九一八事变
  - 西安事变
- **历史评价**：发动西安事变，促成国共合作

### 杨虎城
- **职位**：十七路军总指挥
- **生卒**：1893年11月26日-1949年9月17日
- **籍贯**：陕西蒲城
- **主要事件**：
  - 西安事变
- **历史评价**：与张学良一起发动西安事变

### 李济深
- **职位**：军事委员会桂林办公厅主任
- **生卒**：1885年11月6日-1959年10月9日
- **籍贯**：广西苍梧
- **主要作用**：
  - 在广西组织抗战
  - 团结各方力量
- **历史评价**：坚持抗战到底

### 阎锡山
- **职位**：第二战区司令长官
- **生卒**：1883年10月8日-1960年5月23日
- **籍贯**：山西五台
- **主要作用**：
  - 指挥山西抗战
  - 建立山西新军
- **历史评价**：在山西坚持抗战
