# 薛岳

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 薛岳 |
| 字 | 伯陵 |
| 生卒 | 1896年12月27日-1998年5月3日 |
| 籍贯 | 广东乐昌 |
| 职位 | 第九战区司令长官 |

## 生平简介

薛岳是国民党高级将领，抗日名将。

## 主要贡献

### 长沙会战
- 指挥四次长沙会战
- 取得重大胜利
- 打击了日军嚣张气焰

### 正面战场
- 指挥第九战区作战
- 参与多次重大战役

## 主要战役

- 第一次长沙会战（1939年）
- 第二次长沙会战（1941年）
- 第三次长沙会战（1941年）

## 历史评价

- 抗日名将
- 长沙会战中取得重大胜利
