# 中国军事将领

抗日战争中指挥作战的中国将领。

## 国民党将领

### 李宗仁
- **职位**：第五战区司令长官
- **生卒**：1891年8月13日-1969年1月30日
- **籍贯**：广西临桂
- **主要战役**：
  - 台儿庄战役（1938年）
- **历史评价**：取得抗战以来正面战场最大胜利
