# 文化名人

用笔杆子战斗的文化工作者。

## 文学界

### 鲁迅
- **原名**：周树人
- **生卒**：1881年9月25日-1936年10月19日
- **籍贯**：浙江绍兴
- **代表作**：
  - 《呐喊》
  - 《彷徨》
  - 《阿Q正传》
- **主要贡献**：
  - 用文学作品激励民族精神
  - 揭露社会黑暗，唤醒民众
- **历史评价**：中国现代文学奠基人
