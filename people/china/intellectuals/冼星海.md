# 冼星海

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 冼星海 |
| 生卒 | 1905年6月13日-1945年10月30日 |
| 籍贯 | 广东番禺 |

## 生平简介

冼星海是中国人民音乐家，被誉为"人民音乐家"。

## 主要贡献

### 音乐创作
- 创作大量抗战音乐
- 《黄河大合唱》
- 《生产大合唱》
- 《到敌人后方去》

### 抗战宣传
- 用音乐反映中国人民抗战精神
- 宣传抗日救亡

## 代表作品

- 《黄河大合唱》
- 《生产大合唱》
- 《到敌人后方去》
- 《在太行山上》

## 历史评价

- 人民音乐家
- 抗战音乐代表人物

## 大事年表

| 时间 | 事件 |
|------|------|
| 1905年6月13日 | 出生于广东番禺 |
| 1918年 | 入北京师范大学附属中学 |
| 1926年 | 入北京大学音乐传习所 |
| 1928年 | 入上海国立音乐学院 |
| 1929年 | 赴法国留学 |
| 1935年 | 回到上海 |
| 1936年 | 创作《到敌人后方去》 |
| 1937年 | 全面抗战爆发 |
| 1938年 | 前往延安 |
| 1939年 | 创作《黄河大合唱》 |
| 1940年 | 赴苏联 |
| 1945年10月30日 | 在莫斯科逝世 |
