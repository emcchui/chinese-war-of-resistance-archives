# 日本政界人物

## 首相

### 近卫文麿
- **生卒**：1891年10月12日-1945年12月16日
- **职务**：首相
- **任职时间**：1937-1939年
- **主要罪行**：
  - 发动全面侵华战争
  - 制定"不以国民政府为对手"方针
- **审判结果**：甲级战犯，自杀
