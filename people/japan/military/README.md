# 日本军方人物

侵华日军的军事将领。

## 侵华日军司令官

### 松井石根
- **生卒**：1878年7月27日-1948年12月23日
- **职务**：华中方面军司令官
- **任职时间**：1937年
- **主要罪行**：
  - 指挥南京大屠杀
  - 对南京军民进行大规模屠杀
- **审判结果**：甲级战犯，绞刑

  - 对华侵略战争
- **审判结果**：甲级战犯，终身监禁

## 被击毙将领

### 阿部规秀
- **生卒**：1886年5月1日-1939年11月7日
- **职务**：陆军中将、独立混成第2旅团旅团长
- **击毙时间**：1939年11月7日
- **击毙地点**：河北黄土岭
- **击毙经过**：
  - 在黄土岭战斗中被八路军击毙
  - 是抗战中击毙的日军最高将领
- **历史意义**：打击了日军嚣张气焰

### 松井太久郎
- **生卒**：？-1940年

- **击毙地点**：华北
- **击毙经过**：
  - 在战斗中被八路军击毙
