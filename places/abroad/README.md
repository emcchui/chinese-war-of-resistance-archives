# 国际战场

中国远征军作战的境外战场。

## 缅甸战场

### 仁安羌
- **战略地位**：缅甸油田区
- **主要战役**：仁安羌大捷（1942年4月）
- **战斗经过**：
  - 英军被围困
  - 中国远征军新38师驰援
  - 击溃日军，解救英军7000多人
- **历史意义**：中国军队的国际荣誉之战
- **历史遗迹**：仁安羌大捷纪念碑

### 同古
- **战略地位**：缅甸中部重镇
- **主要战役**：同古保卫战（1942年3月）
- **战斗经过**：
  - 中国远征军第200师坚守同古
  - 与日军激战12天
  - 完成掩护任务后撤退
- **历史意义**：中国远征军入缅作战第一仗
- **历史遗迹**：同古保卫战纪念碑

### 密支那
- **战略地位**：缅甸北部重镇
- **主要战役**：密支那战役（1944年5月-8月）
- **战斗经过**：
  - 中国驻印军进攻密支那
  - 与日军激战3个月
  - 攻克密支那
- **历史意义**：打通中印公路的关键战役
- **历史遗迹**：密支那战役纪念碑

### 八莫
- **战略地位**：缅甸北部城市
- **主要战役**：八莫战役（1944年10月-12月）
- **战斗经过**：
  - 中国驻印军进攻八莫
  - 与日军激战
  - 攻克八莫
- **历史意义**：打通中印公路
- **历史遗迹**：八莫战役纪念碑

### 腊戍
- **战略地位**：缅甸北部城市
- **主要战役**：腊戍战役（1945年）
- **战斗经过**：
  - 中国远征军进攻腊戍
  - 攻克腊戍
- **历史意义**：打通滇缅公路
- **历史遗迹**：腊戍战役纪念碑

## 印度战场

### 兰姆伽
- **战略地位**：印度比哈尔邦城镇
- **主要活动**：中国驻印军训练基地
- **训练内容**：
  - 美式装备训练
  - 丛林作战训练
  - 无线电通讯训练
- **历史意义**：中国军队现代化训练基地
- **历史遗迹**：兰姆伽训练基地旧址

### 雷多
- **战略地位**：印度阿萨姆邦城镇
- **主要活动**：中印公路起点
- **历史意义**：中国驻印军反攻出发地
- **历史遗迹**：雷多公路起点纪念碑

## 重要公路

### 滇缅公路
- **起点**：云南昆明
- **终点**：缅甸腊戍
- **全长**：约1150公里
- **修建时间**：1937年-1938年
- **历史意义**：抗战时期国际援华物资主要通道

### 中印公路
- **起点**：印度雷多
- **终点**：云南昆明
- **全长**：约1700公里
- **修建时间**：1943年-1945年
- **历史意义**：打通国际援华物资通道

### 驼峰航线
- **起点**：印度阿萨姆邦
- **终点**：云南昆明
- **航线长度**：约800公里
- **飞行高度**：4000-6000米
- **历史意义**：抗战时期重要的空中补给线

## 战役统计

### 第一次入缅作战（1942年）
- **时间**：1942年2月-5月
- **兵力**：约10万人
- **主要战役**：同古保卫战、仁安羌大捷
- **结果**：失利撤退

### 第二次入缅作战（1943年-1945年）
- **时间**：1943年10月-1945年3月
- **兵力**：约20万人
- **主要战役**：胡康河谷战役、密支那战役、八莫战役
- **结果**：取得胜利

## 历史意义

1. 打通国际援华通道
2. 歼灭大量日军
3. 提高中国军队国际威望
4. 配合盟军作战
