# 正面战场

国民政府军队作战的主要战场。

## 华北战场

### 北平
- **战略地位**：华北政治、文化中心
- **主要事件**：七七事变（1937年7月7日）
- **沦陷时间**：1937年7月29日
- **战斗经过**：
  - 29军奋起抵抗
  - 副军长佟麟阁、师长赵登禹殉国
  - 因装备劣势，被迫撤退
- **历史遗迹**：卢沟桥、宛平城、中国人民抗日战争纪念馆

### 天津
- **战略地位**：华北经济中心、港口城市
- **主要事件**：天津保卫战
- **沦陷时间**：1937年7月30日
- **战斗经过**：
  - 29军38师奋起抵抗
  - 与日军激战
  - 因孤立无援，被迫撤退
- **历史遗迹**：天津抗战纪念馆

### 太原
- **战略地位**：山西省会，华北重要城市
- **主要战役**：太原会战（1937年9月-11月）
- **沦陷时间**：1937年11月8日
- **战斗经过**：
  - 忻口战役：国共军队协同作战
  - 平型关大捷：八路军取得胜利
  - 太原保卫战：激战后失守
- **历史遗迹**：太原会战纪念馆、平型关大捷纪念馆

### 临汾
- **战略地位**：山西南部重镇
- **主要事件**：临汾沦陷
- **沦陷时间**：1938年2月
- **战斗经过**：
  - 日军沿同蒲路南犯
  - 中国军队节节抵抗
  - 临汾失守
- **历史遗迹**：临汾抗战遗址

### 石家庄
- **战略地位**：华北交通枢纽
- **主要事件**：石家庄沦陷
- **沦陷时间**：1937年10月
- **战斗经过**：
  - 日军沿正太路进攻
  - 中国军队抵抗
  - 石家庄失守
- **历史遗迹**：石家庄抗战纪念馆

## 华东战场

### 上海
- **战略地位**：中国最大的经济中心
- **主要战役**：淞沪会战（1937年8月13日-11月12日）
- **双方兵力**：中国约70万人，日军约30万人
- **沦陷时间**：1937年11月12日
- **战斗经过**：
  - 8月13日：日军进攻闸北
  - 8月14日：中国空军出击
  - 8月23日：日军登陆
  - 10月26日：四行仓库保卫战
  - 11月12日：上海沦陷
- **历史意义**：粉碎了日军"三个月灭亡中国"的计划
- **历史遗迹**：四行仓库、淞沪抗战纪念馆

### 南京
- **战略地位**：国民政府首都
- **主要事件**：南京保卫战、南京大屠杀
- **沦陷时间**：1937年12月13日
- **战斗经过**：
  - 12月7日：蒋介石离开南京
  - 12月9日：日军向南京发起总攻
  - 12月12日：唐生智下令撤退
  - 12月13日：南京沦陷
  - 12月13日起：南京大屠杀
- **历史意义**：南京大屠杀是日军暴行的铁证
- **历史遗迹**：侵华日军南京大屠杀遇难同胞纪念馆

### 徐州
- **战略地位**：津浦线和陇海线交汇点
- **主要战役**：徐州会战（1938年1月-5月）
- **主要战斗**：台儿庄战役
- **沦陷时间**：1938年5月19日
- **战斗经过**：
  - 1月：日军进攻徐州
  - 3月-4月：台儿庄大捷
  - 5月：日军合围徐州
  - 5月19日：徐州沦陷
- **历史意义**：台儿庄大捷是抗战以来正面战场最大胜利
- **历史遗迹**：台儿庄大战纪念馆

### 杭州
- **战略地位**：浙江省会，沪杭甬铁路终点
- **主要事件**：杭州沦陷
- **沦陷时间**：1937年12月24日
- **战斗经过**：
  - 日军从上海向杭州进犯
  - 中国军队抵抗
  - 杭州失守
- **历史遗迹**：杭州抗战纪念馆

## 华中战场

### 武汉
- **战略地位**：华中重镇，"九省通衢"
- **主要战役**：武汉会战（1938年6月-10月）
- **双方兵力**：中国约100万人，日军约30万人
- **沦陷时间**：1938年10月25日
- **战斗经过**：
  - 6月：日军进攻武汉外围
  - 8月：万家岭大捷
  - 10月：日军占领广州
  - 10月25日：武汉沦陷
- **历史意义**：抗战战略防御阶段结束
- **历史遗迹**：武汉会战纪念馆

### 长沙
- **战略地位**：湖南省会，华中战略要地
- **主要战役**：四次长沙会战（1939年-1942年）
- **沦陷时间**：1944年6月19日（第四次）
- **战斗经过**：
  - 第一次长沙会战（1939年9月）：中国胜利
  - 第二次长沙会战（1941年9月）：中国失利
  - 第三次长沙会战（1941年12月）：中国胜利
  - 第四次长沙会战（1944年5月）：长沙沦陷
- **历史意义**：长沙会战是抗战相持阶段重要战役
- **历史遗迹**：长沙会战纪念碑

### 宜昌
- **战略地位**：长江三峡入口，四川门户
- **主要战役**：宜昌战役
- **沦陷时间**：1940年6月12日
- **战斗经过**：
  - 日军进攻宜昌
  - 中国军队抵抗
  - 宜昌失守
- **历史遗迹**：宜昌抗战遗址

### 襄阳
- **战略地位**：鄂北重镇
- **主要战役**：枣宜会战
- **沦陷时间**：1940年5月
- **战斗经过**：
  - 日军进攻襄阳
  - 张自忠将军殉国
  - 襄阳失守
- **历史遗迹**：张自忠将军殉国处

## 华南战场

### 广州
- **战略地位**：华南最大城市，对外贸易中心
- **主要事件**：广州沦陷
- **沦陷时间**：1938年10月21日
- **战斗经过**：
  - 日军从大亚湾登陆
  - 向广州进攻
  - 广州失守
- **历史遗迹**：广州抗战纪念馆

### 桂林
- **战略地位**：广西省会，西南重镇
- **主要战役**：桂柳会战
- **沦陷时间**：1944年11月10日
- **战斗经过**：
  - 日军进攻桂林
  - 中国军队抵抗
  - 桂林失守
- **历史遗迹**：桂林抗战遗址

### 昆明
- **战略地位**：云南省会，大后方重要城市
- **主要事件**：滇缅公路终点，驼峰航线终点
- **历史遗迹**：飞虎队纪念馆
