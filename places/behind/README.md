# 敌后战场

八路军、新四军作战的敌后抗日根据地。

## 陕甘宁边区

### 延安
- **地位**：中共中央所在地，敌后战场指挥中心
- **时间**：1937年1月-1947年3月
- **主要机构**：
  - 中共中央
  - 八路军总部
  - 陕甘宁边区政府
- **主要活动**：
  - 持久战理论
  - 大生产运动
  - 整风运动
- **历史遗迹**：延安革命纪念馆、杨家岭、枣园、凤凰山

### 边区特点
- **政治**：实行民主选举
- **经济**：开展大生产运动
- **文化**：创办抗日军政大学
- **军事**：保卫边区，支援前线

## 晋察冀根据地

### 概况
- **位置**：山西、察哈尔、河北三省交界
- **创建时间**：1937年11月
- **领导人**：聂荣臻
- **面积**：约20万平方公里
- **人口**：约2500万

### 主要地区
- **北岳区**：根据地中心
- **冀中区**：平原游击战
- **冀东区**：敌后游击区
- **平西区**：战略要地

### 主要战役
- **黄土岭战役**（1939年11月）：击毙阿部规秀
- **百团大战**（1940年8月-12月）：破坏交通线
- **反扫荡作战**（1941年-1942年）：保卫根据地

### 历史遗迹
- 晋察冀军区司令部旧址
- 晋察冀边区政府旧址
- 狼牙山五壮士纪念地

## 根据地特点

### 地理特点
- 多位于省际交界处
- 多位于山区或偏远地区
- 便于开展游击战

### 军事特点
- 以游击战为主
- 依靠人民群众
- 建立地方武装

### 政治特点
- 建立抗日民主政权
- 实行减租减息
- 团结各阶层抗日

### 经济特点
- 开展大生产运动
- 实行自给自足
- 支援前线作战
