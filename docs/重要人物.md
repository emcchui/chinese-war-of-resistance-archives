# 抗日战争重要人物

## 中国领导人

### 蒋介石
- **职位**：国民政府军事委员会委员长
- **作用**：领导国民政府抗战，指挥正面战场作战
- **评价**：坚持抗战到底，但战略战术有失误

### 毛泽东
- **职位**：中国共产党中央委员会主席
- **作用**：领导敌后抗战，提出持久战理论
- **评价**：正确领导敌后战场，为抗战胜利奠定基础

### 周恩来
- **职位**：中共中央军委副主席
- **作用**：负责统战工作，推动国共合作
- **评价**：为团结抗战作出重要贡献

## 军事将领

### 朱德
- **职位**：八路军总司令
- **作用**：指挥八路军敌后抗战
- **主要战役**：百团大战
- **评价**：敌后战场主要指挥者

### 彭德怀
- **职位**：八路军副总司令
- **作用**：指挥百团大战等重要战役
- **主要战役**：百团大战
- **评价**：英勇善战，指挥果断

### 李宗仁

- **主要战役**：台儿庄战役

### 张自忠

- **主要战役**：枣宜会战
- **评价**：抗战中牺牲的最高将领之一

### 戴安澜
- **职位**：中国远征军第200师师长
- **作用**：在缅甸作战中壮烈殉国
- **主要战役**：同古保卫战
- **评价**：中国远征军著名将领

### 孙立人
- **职位**：新1军军长
- **作用**：在缅甸作战中取得重大胜利
- **主要战役**：仁安羌大捷
- **评价**：抗日名将，国际威望高
