# 中国抗日战争资料库

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

本仓库系统收集和整理中国抗日战争（1931-1945）的全过程细节资料。

## 目录结构

```
├── people/                # 人物
│   ├── china/             # 中国人物
│   │   ├── leaders/       # 领导人物
│   │   ├── generals/      # 军事将领
│   │   ├── martyrs/       # 殉国英烈
│   │   └── intellectuals/ # 文化名人
│   ├── japan/             # 日本人物
│   │   ├── military/      # 军方人物
│   │   ├── politicians/   # 政界人物
│   │   └── war_criminals/ # 战犯
│   └── other/             # 其他国家人物
│       ├── allies/        # 盟国将领
│       ├── volunteers/    # 国际志愿者
│       └── witnesses/     # 历史见证者
├── places/                # 地点
│   ├── front/             # 正面战场
│   ├── behind/            # 敌后战场
│   └── abroad/            # 国际战场
├── events/                # 事件
│   ├── incidents/         # 事变
│   ├── battles/           # 战役
│   └── massacres/         # 惨案
├── sources/               # 原始资料
│   ├── documents/         # 文献档案
│   ├── videos/            # 影像资料
│   └── oral/              # 口述历史
├── photos/                # 历史照片
├── maps/                  # 战役地图
├── references/            # 参考书目
├── timeline/              # 时间线
└── docs/                  # 综合文档
```
