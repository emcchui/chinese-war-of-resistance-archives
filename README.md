# 中国抗日战争资料库

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

本仓库系统收集和整理中国抗日战争（1931-1945）的全过程细节资料，包括历史事件、重要战役、人物传记、文献资料等，旨在为研究者和历史爱好者提供全面、准确的参考资料。

## 项目简介

抗日战争是中华民族近代以来第一次取得完全胜利的民族解放战争，是中国人民为维护国家主权和民族尊严而进行的伟大斗争。本项目致力于：

- 系统梳理抗日战争的完整历史脉络
- 记录重要历史事件和战役详情
- 收集相关文献资料和历史研究
- 传承伟大的抗战精神

## 主要内容

### 时间线 (timeline/)
- **1931-1937 局部抗战时期**：九一八事变、一二八事变、华北事变等
- **1937-1945 全面抗战时期**：卢沟桥事变、台儿庄战役、武汉会战、百团大战等
- **1942-1945 国际作战**：中国远征军入缅作战

### 重要事件
- 九一八事变（1931年）
- 卢沟桥事变/七七事变（1937年）
- 南京大屠杀（1937年）
- 台儿庄战役（1938年）
- 百团大战（1940年）
- 中国远征军入缅作战（1942-1945年）

### 人物资料
- 军事将领：蒋介石、朱德、彭德怀、李宗仁、张自忠等
- 地方将领：阎锡山、李济深等
- 敌后战场：聂荣臻、贺龙、刘伯承等
- 国际友人：白求恩、柯棣华、拉贝等

### 参考文献
- 政治文献：《论持久战》、《抗日救国十大纲领》等
- 军事文献：对日作战方针等
- 文化文献：《义勇军进行曲》、《黄河大合唱》等
- 新闻文献：《新华日报》、《解放日报》等
