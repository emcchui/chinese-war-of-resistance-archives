# 文献档案

抗日战争的文字资料。

## 政治文献

### 《论持久战》
- **作者**：毛泽东
- **时间**：1938年5月
- **内容**：
  - 分析抗日战争的特点和规律
  - 提出持久战理论
  - 预见抗战三个阶段：战略防御、战略相持、战略反攻
- **意义**：指导全国抗战的纲领性文献

### 《抗日救国十大纲领》
- **发布机构**：中共中央
- **时间**：1937年8月25日
- **内容**：
  - 打倒日本帝国主义
  - 全国军事总动员
  - 全国人民总动员
  - 改革政治机构
  - 抗日外交政策
  - 战时财政经济政策
  - 改善人民生活
  - 抗日教育政策
  - 肃清汉奸卖国贼
  - 抗日民族团结
- **意义**：中共全面抗战路线的体现

### 《抗战建国纲领》
- **发布机构**：国民政府
- **时间**：1938年4月
- **内容**：
  - 确定抗战建国的基本方针
  - 规定军事、政治、经济、外交等方面的政策
- **意义**：国民政府抗战时期的基本国策

### 《中共中央为公布国共合作宣言》
- **发布机构**：中共中央
- **时间**：1937年9月22日
- **内容**：
  - 提出团结抗日、实行民主、改善民生等主张
  - 保证共产党员效力于抗日民族统一战线
- **意义**：标志国共第二次合作正式形成

## 军事文献

### 《对日作战方针》
- **发布机构**：中共中央
- **时间**：1937年8月
- **内容**：
  - 制定对日作战的基本方针
  - 强调独立自主的山地游击战
- **意义**：指导八路军、新四军作战

### 《关于目前党的政策中的几个重要问题》
- **作者**：毛泽东
- **时间**：1940年
- **内容**：
  - 提出敌后抗战的政策
  - 强调抗日根据地建设
- **意义**：指导敌后战场作战

### 《关于抗日根据地土地政策的决定》
- **发布机构**：中共中央
- **时间**：1942年1月
- **内容**：
  - 确定抗日根据地土地政策
  - 实行减租减息
- **意义**：调动农民抗日积极性

## 新闻文献

### 《新华日报》
- **性质**：中共在国统区公开发行的报纸
- **时间**：1938年-1947年
- **地点**：武汉、重庆
- **内容**：
  - 宣传抗战
  - 揭露投降
  - 报道敌后战场
- **意义**：中共在国统区的重要宣传阵地

### 《解放日报》
- **性质**：中共中央机关报
- **时间**：1941年-1947年
- **地点**：延安
- **内容**：
  - 宣传抗战政策
  - 报道敌后战场
  - 指导根据地建设
- **意义**：中共的重要宣传阵地

### 《救亡日报》
- **性质**：上海文化界救亡协会机关报
- **时间**：1937年-1941年
- **地点**：上海、广州、桂林
- **内容**：
  - 宣传抗日救亡
  - 报道抗战动态
- **意义**：文化界抗日救亡的重要阵地

## 文化文献

### 《义勇军进行曲》
- **作曲**：聂耳
- **作词**：田汉
- **时间**：1935年
- **内容**：
  - "起来！不愿做奴隶的人们！"
  - "把我们的血肉，筑成我们新的长城！"
- **意义**：激发全国人民抗战热情，后成为国歌
