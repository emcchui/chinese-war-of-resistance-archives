# 抗日战争时间线

抗日战争（1931年9月18日—1945年9月2日）是中国近代史上一次伟大的民族解放战争。

## 时间线概览

### 第一阶段：局部抗战（1931-1937）

#### 1931年
- **9月18日** - 九一八事变：日本关东军炸毁南满铁路沈阳北郊柳条湖附近的一段路轨，反诬中国军队所为，以此为借口进攻沈阳
- **9月19日** - 日军占领沈阳
- **9月-12月** - 日军迅速占领辽宁、吉林、黑龙江省
- **12月** - 国际联盟派出李顿调查团

#### 1932年
- **1月28日** - 一·二八事变：日军进攻上海，十九路军奋起抵抗
- **3月1日** - 日本扶植成立伪满洲国，溥仪任执政
- **5月5日** - 《淞沪停战协定》签订

#### 1933年
- **1月3日** - 日军占领山海关
- **3月** - 日军占领热河省
- **5月** - 《塘沽协定》签订

#### 1935年
- **6月** - 《何梅协定》签订
- **8月1日** - 中共发表《八一宣言》，呼吁停止内战，一致抗日
- **12月9日** - 一二·九运动：北平学生抗日救亡运动

#### 1936年
- **12月12日** - 西安事变：张学良、杨虎城扣押蒋介石，逼蒋抗日

### 第二阶段：全面抗战（1937-1945）

#### 1937年
- **7月7日** - 卢沟桥事变（七七事变）：全面抗战爆发
- **7月29日** - 北平沦陷
- **7月30日** - 天津沦陷
- **8月13日** - 八一三事变：日军进攻上海
- **8月22日** - 红军改编为国民革命军第八路军
- **9月22日** - 国共第二次合作正式形成
- **9月25日** - 平型关大捷：八路军取得抗战以来第一个大胜利
- **11月12日** - 上海沦陷
- **11月20日** - 国民政府迁都重庆
- **12月13日** - 南京沦陷，南京大屠杀开始
- **12月** - 南京大屠杀：日军屠杀中国军民30万人以上

#### 1938年
- **2月** - 台儿庄战役开始
- **4月7日** - 台儿庄大捷：中国军队取得抗战以来正面战场最大胜利
- **6月-10月** - 武汉会战
- **10月25日** - 武汉沦陷
- **10月21日** - 广州沦陷
- **12月** - 汪精卫叛国投敌

#### 1939年
- **5月** - 随枣会战
- **9月** - 第一次长沙会战
- **11月** - 桂南会战开始

#### 1940年
- **3月30日** - 汪精卫伪国民政府在南京成立
- **5月** - 枣宜会战，张自忠将军殉国
- **8月20日** - 百团大战开始
- **10月** - 百团大战结束

#### 1941年
- **1月** - 皖南事变
- **3月** - 上高会战
- **12月8日** - 太平洋战争爆发，中国对日宣战
- **12月** - 香港沦陷

#### 1942年
- **1月** - 中国远征军入缅作战
- **4月** - 仁安羌大捷
- **5月** - 中国远征军失利，撤退

#### 1943年
- **10月** - 中国驻印军开始反攻
- **11月** - 常德会战

#### 1944年
- **4月** - 豫湘桂战役开始
- **6月** - 衡阳保卫战
- **10月** - 湘西会战开始
- **12月** - 豫湘桂战役结束

#### 1945年
- **4月** - 湘西会战（芷江战役）
- **5月** - 湖南、广西光复
- **8月6日** - 美国在广岛投下原子弹
- **8月8日** - 苏联对日宣战
- **8月9日** - 美国在长崎投下原子弹
- **8月15日** - 日本宣布无条件投降
- **9月2日** - 日本正式签署投降书，抗日战争胜利结束
- **9月3日** - 中国抗日战争胜利纪念日
- **9月9日** - 中国战区日军投降签字仪式在南京举行

## 重要战役

1. 平型关大捷（1937年9月）
2. 台儿庄战役（1938年3-4月）
3. 武汉会战（1938年6-10月）
4. 长沙会战（1939-1942年）
5. 百团大战（1940年8-10月）
6. 中国远征军入缅作战（1942-1945年）

## 重要人物

- 蒋介石：国民政府军事委员会委员长
- 朱德：八路军总司令
- 彭德怀：八路军副总司令，百团大战指挥者
- 李宗仁：台儿庄战役指挥者
- 张自忠：枣宜会战殉国的陆军上将
- 戴安澜：中国远征军200师师长，殉国于缅甸

## 历史意义

1. 抗日战争是中国近代以来第一次取得完全胜利的民族解放战争
2. 洗雪了近代以来的民族耻辱
3. 为世界反法西斯战争作出重要贡献
4. 提高了中国的国际地位
