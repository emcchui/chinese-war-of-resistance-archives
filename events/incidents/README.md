# 事变

日本侵略中国的重要事件。

## 九一八事变

### 基本信息
- **时间**：1931年9月18日
- **地点**：沈阳
- **别称**：柳条湖事件
- **结果**：日军占领东北三省

### 事件背景
- 1927年东方会议：日本制定侵华方针
- 世界经济危机：日本国内矛盾激化
- 关东军不断制造事端

### 事件经过
- **9月18日夜**：日军炸毁南满铁路柳条湖附近路轨
- **22时20分**：日军炮击北大营
- **22时30分**：日军步兵进攻北大营
- **9月19日凌晨**：日军占领北大营
- **9月19日上午**：日军占领沈阳城

### 东北军反应
- 张学良执行不抵抗政策
- 东北军大部分未进行有效抵抗
- 少数官兵自发抵抗

### 后续发展
- 9月19日：日军占领长春
- 9月21日：日军占领吉林
- 11月19日：日军占领齐齐哈尔
- 1932年1月3日：日军占领锦州
- 1932年2月5日：日军占领哈尔滨

### 历史意义
1. 中国人民抗日战争的起点
2. 揭开了世界反法西斯战争的序幕
3. 促进了中华民族的觉醒

## 一·二八事变

### 基本信息
- **时间**：1932年1月28日
- **地点**：上海
- **别称**：淞沪抗战
- **结果**：签订《淞沪停战协定》

### 事件背景
- 日本为转移国际视线
- 扶植伪满洲国
- 在上海制造事端

### 事件经过
- **1月28日夜**：日军进攻上海闸北
- **1月29日**：十九路军奋起抵抗
- **2月-3月**：双方激战
- **3月3日**：日军增援，中国军队撤退
- **5月5日**：签订停战协定

### 主要战斗
- 闸北保卫战
- 吴淞口保卫战
- 淞沪铁路战斗

### 历史意义
1. 中国军队首次大规模抵抗日军
2. 显示了中国军队的抗战决心
3. 打破了日军不可战胜的神话

## 华北事变

### 基本信息
- **时间**：1935年
- **地点**：华北
- **结果**：签订《何梅协定》《秦土协定》

### 事件经过
- **5月**：河北事件
- **6月**：察东事件
- **6月**：签订《何梅协定》
- **7月**：签订《秦土协定》

### 主要内容
- 日本策动华北自治
- 迫使国民政府撤军
- 华北主权遭到严重损害

### 历史意义
1. 日本侵华的重要步骤
2. 促进了抗日救亡运动的发展
3. 为全面侵华做准备

## 西安事变

### 基本信息
- **时间**：1936年12月12日
- **地点**：西安
- **别称**：双十二事变
- **结果**：国共合作初步形成

### 事件背景
- 日本加紧侵华
- 蒋介石坚持"攘外必先安内"
- 张学良、杨虎城逼蒋抗日

### 事件经过
- **12月12日凌晨**：张学良、杨虎城扣押蒋介石
- **12月12日**：通电全国，提出八项主张
- **12月13日**：中共派周恩来赴西安
- **12月24日**：蒋介石接受停止内战、联共抗日
- **12月25日**：蒋介石返回南京

### 历史意义
1. 国内战争基本结束
2. 国共合作初步形成
3. 抗日民族统一战线初步形成

## 七七事变

### 基本信息
- **时间**：1937年7月7日
- **地点**：北平西南卢沟桥
- **别称**：卢沟桥事变
- **结果**：全面抗战爆发

### 事件背景
- 日本在华北不断制造事端
- 卢沟桥是北平对外交通要道
- 中国军队在此驻防

### 事件经过
- **7月7日夜**：日军演习中声称士兵失踪
- **7月8日凌晨**：日军炮击宛平城
- **7月8日**：29军奋起抵抗
- **7月9日**：双方达成临时停火协议
- **7月10日**：日军再次发动进攻
- **7月25日**：日军进攻廊坊
- **7月26日**：日军向29军发出最后通牒
- **7月28日**：日军发动全面进攻
- **7月29日**：北平沦陷
- **7月30日**：天津沦陷

### 主要战斗
- 卢沟桥战斗
- 宛平城保卫战
- 南苑战斗

### 历史意义
1. 全面抗战的起点
2. 国共第二次合作形成
3. 中华民族团结抗战

## 八一三事变

### 基本信息
- **时间**：1937年8月13日
- **地点**：上海
- **别称**：淞沪会战开始
- **结果**：上海沦陷

### 事件经过
- **8月13日**：日军进攻上海闸北
- **8月14日**：中国空军出击
- **8月23日**：日军登陆
- **10月26日**：四行仓库保卫战
- **11月12日**：上海沦陷

### 历史意义
1. 粉碎了日军"三个月灭亡中国"的计划
2. 显示了中国军队的抗战决心
3. 为工业内迁争取了时间
