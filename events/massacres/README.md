# 惨案

日军在中国制造的屠杀惨案。

## 南京大屠杀

### 基本信息
- **时间**：1937年12月13日-1938年1月
- **地点**：南京及周边地区
- **遇难人数**：30万人以上
- **性质**：日军有组织、有计划的大规模屠杀

### 事件背景
- 1937年12月，日军占领南京
- 大量军民未能及时撤离
- 日军在南京进行大规模屠杀

### 屠杀暴行

#### 集体屠杀
- **汉中门外**：2000余人遇难
- **中山码头**：5000余人遇难
- **草鞋峡**：5万余人遇难
- **下关**：大量难民遇难

#### 零散屠杀
- 日军在南京城内随意屠杀平民
- 见人就杀，见物就抢
- 屠杀手段极其残忍

#### 强奸暴行
- 大量妇女遭到强奸
- 许多妇女被强奸后杀害
- 暴行持续数周

#### 抢掠焚烧
- 日军大肆抢掠财物
- 焚烧房屋、破坏城市
- 南京城内一片废墟

### 难民区
- 外国侨民建立南京安全区
- 约25万难民进入安全区避难
- 但安全区仍未能完全保护难民

### 历史证据
- 幸存者证言
- 外国侨民记录
- 东京审判记录
- 马吉牧师拍摄的影像

### 战后审判
- 远东国际军事法庭：判定南京大屠杀是战争罪行
- 松井石根被判有罪并执行绞刑
- 谷寿夫被判有罪并执行枪决

### 历史意义
1. 向世界揭露日军侵华暴行
2. 提醒国人勿忘国耻
3. 警示后人珍惜和平

### 纪念设施
- 侵华日军南京大屠杀遇难同胞纪念馆
- 哭墙（刻有遇难者姓名）
- 国家公祭日（12月13日）

## 潘家峪惨案

### 基本信息
- **时间**：1941年1月25日
- **地点**：河北丰润潘家峪
- **遇难人数**：1230人
- **性质**：日军报复性屠杀

### 事件经过
- 日军包围潘家峪村
- 将村民驱赶到村西大坑
- 用机枪扫射、手榴弹轰炸
- 焚烧房屋，毁尸灭迹

### 历史意义
- 日军暴行的铁证
- 激发了当地人民的抗日斗志

### 纪念设施
- 潘家峪惨案纪念馆
- 潘家峪惨案纪念碑
