# 国外战役

抗日战争中中国军队在境外进行的主要战役。

## 缅甸战场

| 战役 | 时间 | 地点 | 结果 |
|------|------|------|------|
| 仁安羌大捷 | 1942年4月 | 缅甸 | 解救英军 |
| 密支那战役 | 1944年5月-8月 | 缅甸 | 中国胜利 |
| 八莫战役 | 1944年10月-12月 | 缅甸 | 中国胜利 |

## 战役意义

- 打通国际援华通道
- 提高了中国军队国际威望
- 配合盟军作战

## 重要公路

- **滇缅公路**：连接中国和缅甸
- **中印公路**：连接印度和中国
- **驼峰航线**：空中补给线
