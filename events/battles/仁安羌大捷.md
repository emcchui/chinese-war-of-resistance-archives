# 仁安羌大捷

## 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 1942年4月 |
| 地点 | 缅甸仁安羌 |
| 参战部队 | 中国远征军新38师 |
| 结果 | 中国胜利 |
| 意义 | 中国军队的国际荣誉之战 |

## 战役背景

### 战略态势
- 中国远征军入缅作战
- 英军被日军包围
- 中国远征军驰援

### 双方部署
- **中国**：新38师，约8000人
- **日军**：约7000人

## 战役经过

### 英军被围
- **4月**：英军在仁安羌被日军包围
- 英军约7000人被困

### 中国远征军驰援
- **4月**：新38师奉命驰援
- 向仁安羌进攻

### 解围作战
- **4月**：新38师击溃日军
- 解救被围英军7000多人

## 战役成果

### 军事成果
- 击溃日军
- 解救英军7000多人
- 缴获大量武器装备

### 战略意义
- 中国军队的国际荣誉之战
- 提高了中国军队国际威望
- 增强了中英军事合作
