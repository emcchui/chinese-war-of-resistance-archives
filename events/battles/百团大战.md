# 百团大战

## 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 1940年8月20日-12月5日 |
| 地点 | 华北 |
| 参战部队 | 八路军105个团，约20万人 |
| 结果 | 中国胜利 |
| 意义 | 抗战期间八路军最大规模战役 |

## 战役背景

### 战略形势
- 日军对华北抗日根据地进行"囚笼政策"
- 修建大量铁路、公路
- 八路军决定发动大规模破袭战

### 战役目的
- 破坏日军交通线
- 拔除日军据点
- 打击日军嚣张气焰

## 战役经过

### 第一阶段：交通破袭战（8月20日-9月10日）
- **8月20日**：八路军105个团同时出击
- 重点破袭正太铁路
- 切断日军交通线
- 摧毁大量据点

### 第二阶段：拔除据点（9月20日-10月5日）
- 继续破袭交通线
- 拔除日军深入根据地的据点
- 扩大战果

### 第三阶段：反"扫荡"作战（10月6日-12月5日）
- 日军调集重兵进行报复性"扫荡"
- 八路军进行反"扫荡"作战
- 粉碎日军进攻

## 主要战斗

### 正太铁路破袭战
- **时间**：1940年8月20日-9月10日
- **经过**：八路军破袭正太铁路
- **意义**：切断日军交通线

### 关家垴战斗
- **时间**：1940年10月
- **经过**：八路军围攻日军
- **意义**：打击日军嚣张气焰

## 战役成果

### 军事成果
- 破坏铁路470多公里
- 破坏公路1500多公里
- 拔除据点2900多个
- 歼灭日伪军4.5万多人

### 战略意义
- 打破了日军"囚笼政策"
- 严重削弱了日军在华北的统治
- 提振了全国军民抗战信心

## 历史意义

1. 打破了日军"囚笼政策"
2. 重创日军在华北的统治
3. 提振了全国军民抗战信心
4. 提高了八路军的威望
