# 太原会战

## 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 1937年9月-11月 |
| 地点 | 山西 |
| 主要战役 | 平型关大捷、忻口战役 |
| 结果 | 太原沦陷 |
| 意义 | 八路军取得抗战以来第一个大胜利 |

## 会战背景

### 战略态势
- 日军向山西进攻
- 太原是山西省会
- 中国决心保卫太原

### 双方部署
- **中国**：第二战区，阎锡山指挥
- **日军**：第5师团等

## 会战经过

### 第一阶段：平型关大捷（9月）
- **9月25日**：八路军115师取得平型关大捷
- 歼灭日军1000多人

### 第二阶段：忻口战役（10月）
- **10月**：日军进攻忻口
- 中国军队顽强抵抗
- 激战数周

### 第三阶段：太原保卫战（11月）
- **11月**：日军进攻太原
- 太原失陷

## 主要战斗

### 平型关大捷
- **时间**：1937年9月25日
- **部队**：八路军115师
- **经过**：伏击日军，歼灭1000多人
- **意义**：抗战以来第一个大胜利

### 忻口战役
- **时间**：1937年10月
- **经过**：中国军队顽强抵抗
- **意义**：消耗日军有生力量

## 战役结果

### 中国方面
- 太原沦陷
- 伤亡约10万人

### 日军方面
- 伤亡约2万人

## 历史意义

1. 平型关大捷是抗战以来第一个大胜利
2. 打破了日军不可战胜的神话
3. 提振了全国军民抗战信心
4. 提高了八路军的威望
