# 台儿庄战役

## 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 1938年3月-4月 |
| 地点 | 山东台儿庄 |
| 双方兵力 | 中国约29万人，日军约5万人 |
| 结果 | 中国胜利 |
| 意义 | 抗战以来正面战场最大胜利 |

## 战役背景

### 战略态势
- 日军企图打通津浦线
- 台儿庄是徐州的门户
- 战略位置重要

### 双方部署
- **中国**：第五战区司令长官李宗仁指挥
- **日军**：第5师团和第10师团

## 战役经过

### 第一阶段：外围战斗（3月14日-3月24日）
- **3月14日**：日军进攻滕县
- **3月17日**：滕县失陷，王铭章殉国
- **3月20日**：日军占领韩庄
- **3月23日**：日军进攻台儿庄

### 第二阶段：台儿庄攻防战（3月24日-4月3日）
- **3月24日**：日军向台儿庄发动进攻
- **3月27日**：日军突入台儿庄
- **3月28日**：中国军队与日军展开巷战
- **3月31日**：中国军队包围台儿庄日军

### 第三阶段：反攻（4月3日-4月6日）
- **4月3日**：中国军队发起全面反攻
- **4月6日**：日军大部被歼

## 主要战斗

### 滕县保卫战
- **时间**：1938年3月14日-17日
- **部队**：第122师
- **指挥官**：王铭章
- **经过**：王铭章率部坚守滕县，壮烈殉国
- **意义**：为台儿庄战役争取时间

### 台儿庄攻防战
- **时间**：1938年3月24日-4月6日
- **经过**：中国军队与日军展开激烈巷战
- **意义**：歼灭日军主力

## 战役结果

### 中国方面
- 取得胜利
- 伤亡约5万人

### 日军方面
- 伤亡约1万人
- 被歼灭大部

## 历史意义

1. 抗战以来正面战场最大胜利
2. 打击了日军嚣张气焰
3. 增强了全国人民抗战信心
4. 提高了中国军队国际威望
