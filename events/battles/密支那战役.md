# 密支那战役

## 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 1944年5月-8月 |
| 地点 | 缅甸密支那 |
| 参战部队 | 中国驻印军 |
| 结果 | 中国胜利 |
| 意义 | 打通中印公路的关键战役 |

## 战役背景

### 战略态势
- 中国驻印军反攻缅甸
- 密支那是缅甸北部重镇
- 打通中印公路的关键

### 双方部署
- **中国**：中国驻印军
- **日军**：第18师团

## 战役经过

### 进攻准备
- **5月**：中国驻印军进攻密支那
- 与日军激战

### 激战过程
- **5月-8月**：与日军激战3个月
- 战斗惨烈

### 攻克密支那
- **8月**：攻克密支那
- 歼灭日军大部

## 战役成果

### 军事成果
- 攻克密支那
- 歼灭日军大部
- 打通中印公路

### 战略意义
- 打通中印公路
- 为后续反攻奠定基础
- 提高了中国军队国际威望

## 历史意义

1. 打通中印公路的关键战役
2. 为后续反攻奠定基础
3. 提高了中国军队国际威望
4. 展现了中国军队的战斗力
